Rustix is a high-performance, Vulkan-powered game engine and editor for Linux. Designed for developers who want control, speed, and a clean architecture.

Get Started View on GitHub
$ git clone https://github.com/tigerlero/Rustix.git
$ cd Rustix
$ cargo run -p rustix-runtime # Launch the editor

Built for Serious Development

A modular engine with an editor-first workflow, real-time rendering, and an ECS core designed to scale.

⚡

Vulkan 1.3 Renderer

Dynamic rendering with gpu-allocator, descriptor sets, push constants, and a staging buffer ring for async uploads.

🧩

Archetypal ECS

hecs-powered ECS with query filters, a component registry, dynamic bundles, and deferred command buffers.

🛠️

Integrated Editor

Edit scenes, place entities, adjust transforms, and manage projects — all inside the runtime with an egui interface.

🔺

Procedural Geometry

Built-in generators for cubes, spheres, toruses, icospheres, capsules, and terrain — ready to render immediately.

💡

Lighting & Shadows

Directional, point, and spot lights with shadow map support and a configurable clear-color pipeline.

🦀

Pure Rust Stack

No C++ glue. From windowing to GPU commands, everything is Rust — with full cargo check and cargo test coverage.

Try It in Minutes

Clone, build, and run. The editor launches out of the box on Linux with a Vulkan-capable GPU.

⚡ Quick Start

cargo run -p rustix-runtime

That's it. The editor opens with a default scene, project management, and a full inspector panel.

📋 Requirements

  • Rust 1.95+ stable
  • Vulkan 1.3 capable GPU
  • Linux (Wayland / X11)
  • Vulkan drivers installed
