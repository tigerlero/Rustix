Engine Assets

Built-in assets, procedural geometry, shaders, audio, and fonts bundled with Rustix Engine.

🎮

3D Models & Meshes

🧩

Procedural Geometry

Built-in primitives generated at runtime via mesh::procedural. No external files required.

  • Cube12 triangles
  • UV Sphereconfigurable slices/stacks
  • Icospheresubdivision-based
  • Torusmajor/minor rings
  • Planeconfigurable subdivisions
📦

GLTF Import

Runtime mesh and material import from .glb / .gltf files with texture support.

  • Meshesindexed geometry
  • MaterialsPBR base color, metallic, roughness
  • Normalsper-vertex or per-face
⛰️

Terrain System

Procedural heightmap terrain with configurable parameters and mesh baking.

  • Heightmapnoise-based generation
  • Terrain meshchunked LOD
🎨

2D Sprites & Textures

🌍

Bindless Textures

GPU-driven texture array with bindless descriptor sets. Albedo, normal, metallic, roughness, AO maps.

  • AlbedoRGBA base color
  • Normaltangent-space normals
  • ORMOcclusion / Roughness / Metallic
🖼️

2D Sprite Pipeline

Orthographic sprite rendering with texture atlases, UV animation, and per-sprite transform.

  • Sprite batchingvertex buffer-based
  • Atlas supporttexture array binding
🌟

Shadow Maps

Dynamic shadow textures generated each frame for directional, point, and spot lights.

  • CSMcascaded shadow maps
  • Point shadowcube depth maps
  • Spot shadowperspective depth
